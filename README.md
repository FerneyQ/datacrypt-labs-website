## 🚀 Despliegue

3. ¡Sitio live en 30 segundos!

### Opción 2: GitHub Pages
1. Subir código a repositorio GitHub
2. Activar Pages en Settings
3. Sitio disponible en GitHub.io

1. Subir archivos vía FTP
